Nonprofit Success Pack API (Application Programming Interface)

The public NPSP API is intended for external use outside the managed package, and includes classes that expose functionality around:

  • Data Import
  • Error Handling
  • Table-Driven Trigger Management (TDTM)
  • Custom Settings
  • Record Types

The classes involved with the public interface include:

  • BDI_DataImport_API
  • CallableApi
  • ERR_Handler_API
  • TDTM_Config_API
  • UTIL_CustomSettings_API
  • UTIL_RecordTypes_API
